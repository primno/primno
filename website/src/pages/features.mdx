# Features

Primno provide modern development features like:

- **Component-based architecture (CBA)**: « Plug and play » behavior. Slicing and applying behaviors on one or multiple forms
- **Dependency injection**: inject services in your components
- **Local development and reloading without deployment**: develop and test faster with reloading on save. No need to upload your webresource.
- **Fast and easy deployment**: upload your project to Dynamics 365 in one command line
- **Strongly typed**: detect errors with Typescript before execution
- **Bundling**: create a single web resource
- **Debugging**: debug your code directly in Dynamics 365 with included source maps
