# Configuration
The configuration of Primno is separated in 2 files available in the root of the project:

| File | Name | Description |
| --- | --- | --- |
| `primno.json` | [Workspace configuration](./workspace.md) | Define the project name, the build folder, where to deploy the entry point, etc. |
| `primno.env.json` | [Environments](./environment.md) of Power Apps | Contains the connection strings for deploying to Power Apps. |
