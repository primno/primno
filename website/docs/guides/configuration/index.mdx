# Configuration
The configuration of Primno is separated in 2 files available in the root of the project:

- `primno.json`: [Workspace configuration](./workspace).

    Define the project name, the build folder, where to deploy the entry points, etc.

- `primno.env.json`: [Environments](./environment) of Power Apps.

    Contains the connection strings for deploying to Power Apps.
